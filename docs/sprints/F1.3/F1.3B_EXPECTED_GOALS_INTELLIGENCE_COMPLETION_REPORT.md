## 3. New xG Football Features

| Feature | Source | Notes |
|---|---|---|
| `xgAttackQualityHome` / `Away` | EXPECTED_GOALS `xg` | [0,100]; absent → omitted |
| `xgDefenseQualityHome` / `Away` | EXPECTED_GOALS `xga` | Higher = stronger defense |
| `xgDominance` | Home xG − Away xG | Signed; both sides required |
| `finishingEfficiencyHome` / `Away` | Goals rate + xG | 50 = in line with xG |

Window preference: `overall` → `last10` → `last5` → `recent` → `fixture` → venue window.

Attack/defense ratings also prefer true EXPECTED_GOALS xG/xGA over zeroed STATISTICS fields when present.
