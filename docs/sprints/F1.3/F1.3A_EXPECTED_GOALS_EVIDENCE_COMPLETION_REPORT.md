## 4. New xG Evidence mapped

Evidence type: `EXPECTED_GOALS`

Each record preserves:

- Provider provenance (`source`, `sourceId`, `method`, collector)
- Timestamp (`observedAt`, Evidence intake timestamps)
- Competition / season (when supplied)
- Team identity + home/away side
- Rolling-window / fixture scope (`overall` ┬╖ `home` ┬╖ `away` ┬╖ `recent` ┬╖ `last5` ┬╖ `last10` ┬╖ `fixture`)
- Metric set (only present keys): `xg`, `xga`, `nonPenaltyXg`, `nonPenaltyXga`, `expectedPoints`, `expectedGoalDifference`
