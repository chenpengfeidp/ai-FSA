# PROJECT BIBLE

## Mission
Build an AI Football Analysis Platform, not a prediction toy.

## Core Principles
1. Long-term improvement over single-match accuracy.
2. Evidence before intuition.
3. Separate facts, market signals and inference.
4. Every analysis must be reviewable.
5. Rules require sample size and confidence.
6. AI explains why, not only what.

## Engines
- **Prompt Engine** — composes reproducible, versioned AI requests.
- **Knowledge Engine** — governs and retrieves approved domain knowledge.
- **Rule Engine** — governs rules and computes deterministic per-snapshot findings.
- **Case Engine** — governs and retrieves reviewed historical analogies.
- **Review Engine** — performs post-match assessment and proposes governed learning.
- **Evaluation Engine** — defines assessment policy and produces reproducible quality and release reports.
- **Statistics Engine** — deterministically computes versioned metrics and rebuildable projections.
